Proyecto Librería Online 

Este es un proyecto desarrollado con HTML, CSS y JavaScript, simula un carrito de compras para una librería en línea donde los usuarios pueden agregar libros al carrito.

Herramientas utilizadas

  • JavaScript (DOM): Manipulación de la interfaz para agregar, eliminar y actualizar libros en el carrito.
  • localStorage: Persistencia de los datos del carrito entre sesiones.
  • CSS: Diseño atractivo con estilos modernos.
  • Bootstrap 5: Diseño atractivo de la página en general

Proyecto Control de Gastos con React  

Este es un Planificador de Gastos desarrollado en React que permite a los usuarios gestionar su presupuesto, registrar gastos y visualizar el porcentaje gastado mediante una barra de progreso circular. Incluye funcionalidades como almacenamiento en localStorage, filtrado de gastos por categoría y edición/eliminación de registros.

Herramientas utilizadas

  • useState: Manejo del estado de los gastos, presupuesto y modales.
  • useEffect: Sincronización con localStorage y actualización de valores.

Proyecto Citas React  

Este es un proyecto desarrollado con React.js, que permite a los usuarios registrar, editar y eliminar citas de mascotas. Los datos se almacenan en localStorage para persistencia entre sesiones.

Herramientas utilizadas

  • useState: Manejo del estado de los pacientes y del formulario.
  • useEffect: Persistencia de los datos en localStorage y actualización en la interfaz.
  • Props: Comunicación entre componentes.
  • Tailwind CSS: Estilos y diseño responsivo.

Proyecto Buscador de Autos  

Esta página es un buscador de autos que permite a los usuarios filtrar vehículos según múltiples criterios, como marca, año, precio, número de puertas, tipo de transmisión y color. La interfaz está diseñada con HTML y CSS.

Herramientas utilizadas

  • Manipulación del DOM: Se usan document.querySelector() para obtener elementos y appendChild() para insertar contenido dinámico.
  • Eventos: Se emplean EventListeners para actualizar el objeto datosBusqueda al seleccionar opciones y DOMContentLoaded para ejecutar funciones al cargar la página.
  • Objetos: datosBusqueda almacena los filtros seleccionados por el usuario.
  • Arreglos y Métodos: Se usa filter() para filtrar autos y forEach() para mostrar los resultados en el DOM.
  • Funciones:
    • mostrarAutos(): Muestra todos los autos.
    • llenarSelectYear(): Genera los años en el formulario.
    • filtrarAuto(): Filtra autos según los criterios.
